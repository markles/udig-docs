Natural Earth

The Natural Earth project provides public domain raster and vector mapping data.

If you would like to refresh this dataset from the Natural Earth project:

  • Natural Earth Downloads

The SLD styles used have been exported from QGIS and cleaned up by hand.

Hotlink

This sample data also includes a hotlink.shp dataset used to support hotlink and document tutorials. This dataset was generated by uDig from the natural earth admin boundary layer by using a filter to select only the states in Australia, and then applying the Transform operation to reduce the dataset to a small number of columns.
